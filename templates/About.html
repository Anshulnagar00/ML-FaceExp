This Web App is built using technologies & API's such as Tensorflow, Flask ,Open-CV, JQuery ,Bootstrap-4 and many more. In this project my all skills are applied such as front-end and back-end web development using python flask , jquery, javascript ,Bootstrap-4 and html-5. Creating Training and Deploying Machine Learning models using Tensorflow and keras API's. Image Processing and Faces detection using OpenCV and MTCNN-classifier.

WorkFlow of App

  1. Resize picked image using javascript
  2. Send the resized image to server via POST method
  3. Store the recieved image in server then preprocess the image
  4. Extract cropped images of faces using OpenCV av and MTCNN-classifier
  5. process the cropped images for the ML Model
  6. fetch the images into CNN model and recieve the results
  7. Send the results with cropped images of faces back through POST method
  8. Display the results by manipulating DOM by javascript
